# Spoider Finds

A hand-curated affiliate storefront for aesthetic Amazon finds — desk setups, fashion, and accessories. Built from scratch with HTML, CSS, and vanilla JavaScript, hosted on GitHub Pages.

## Structure
- `index.html` — homepage
- `about.html`, `contact.html`, `privacy.html`, `affiliate-disclosure.html` — supporting pages
- `404.html` — custom not-found page
- `style.css` — all styling
- `script.js` — nav toggle + scroll effects
- `assets/` — images, icons, fonts

## Live site
https://spoider07.github.io/spoider-finds/
